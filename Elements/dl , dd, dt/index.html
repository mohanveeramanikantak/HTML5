DD TAG

Description, definition, or value, part of a term- description group in a description list. The HTML "dd" element (HTML Description Element) indicates the description of a term in a description list (dl) element. This element can occur only as a child element of a description list and it must follow a dt element. Attributes (modifiers) Global attributes Code example
